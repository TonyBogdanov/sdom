# __construct( mixed $content = NULL )
Dom constructor. Create new collection from the specified content. If the content is NULL, create empty collection. If the content is a Dom instance, copy the nodes into the new collection. If the content is a NodeInterface, add it to the collection. If the content is a Token, convert it to NodeInterface and add it to the collection. If the content is a TokenCollection, convert it to NodeInterface instances and add them to the collection. If the content is a string, parse as HTML and treat as TokenCollection.
